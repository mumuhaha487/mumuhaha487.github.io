通过python用超级简单和基本的方法实现求导数

Jun 11, 2023
1 min read

直接利用数学中导数的定义就行

代码实现就是

MIN=1e-9            #&#x4E00;&#x4E2A;&#x8DB3;&#x591F;&#x5C0F;&#x7684;&#x6570;

import numpy as np
def f_yuan(x_2):
    y_2=x_2**2        #&#x8FD9;&#x91CC;&#x7684;&#x51FD;&#x6570;&#x53EF;&#x4EE5;&#x6539;
    return y_2
def F_daoshu_yuan(x_1):
    a_1=(f_yuan(x_1+MIN)-f_yuan(x_1))/MIN
    return a_1

A_1=F_daoshu_yuan(1)
print(A_1)
print("*******************************************")
A_2=F_daoshu_yuan(10)
print(A_2)
print("*******************************************")
A_3=F_daoshu_yuan(1000)
print(A_3)
print("*******************************************")
A_4=F_daoshu_yuan(10000)
print(A_4)
print("*******************************************")

优缺点

优点是泛用性非常好，也十分好理解，在合适的范围，函数可以变得不规则，但是依旧可以求出导数的大约值，并不会与真实值差的很多

缺点很明显在数据十分大的时候误差也会越来越大，原因就是MIN的值还是太大了。
